How to run the query

For ID temporal Query, you can insert the taxi Id, Time Start and Time end from the input example
For Path temporal Query, you can insert the Time Start and Time end from the input example

Input Example:
1. ID Temporal Query
	- Taxi Id      : 20000596
	- Time Start : 01-07-2013 00:00:54 AM
	- Time end  : 01-07-2013 01:22:38 AM
2. Path Temporal Query
	- Time Start : 01-07-2013 00:00:54 AM
	- Time end  : 01-07-2013 01:22:38 AM
